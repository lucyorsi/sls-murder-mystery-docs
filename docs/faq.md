
# FAQ

**Do I have to play?**  
No. This is a Halloween party first — you can just mingle.

**Who do I ask if I’m confused?**  
Any **host**.

**Can I use potions in guest vs guest fights?**  
No — potions only work against **hosts**.

**Do we heal between fights?**  
Yes — everyone heals to **full HP** after each fight.
